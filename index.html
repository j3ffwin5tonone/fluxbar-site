FluxBar

Minimal real-time monitoring for macOS

Monitor CPU usage, memory pressure, and network activity directly in your macOS menu bar. Fast, lightweight, privacy-focused — no telemetry, no cloud services, no user tracking.

Privacy First

All metrics are processed locally on your Mac. FluxBar never sends any data to external servers.

Detailed Monitoring

Per-core CPU usage with temperature, memory pressure tracking, top process monitoring, and network speeds.

Highly Customizable

Choose which metrics to display, customize icon styles (SF Symbols, Emoji, Dots, Bars, None), color themes, text sizes, and set refresh intervals (1, 2, or 5 seconds).

Native macOS Design

Built with SwiftUI and optimized for macOS 13.0+. Works seamlessly on both Intel and Apple Silicon Macs.

Data Export

Export system metrics to CSV or JSON for analysis. Optional continuous logging for long-term monitoring.

Lightweight

Runs quietly in the background using minimal resources without slowing down your system.
